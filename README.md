# Bank Management System (Java)

## 🚀 Features
- Account creation & login system
- Deposit and withdrawal
- Transaction history tracking
- File handling (data persistence)
- GUI using Java Swing

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- File Handling
- OOP Concepts

## ▶️ How to Run
1. Compile: javac *.java
2. Run: java BankGUI
